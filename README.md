# Qr Code
A widget that can be added to any website; creates a qr code that can be used to view the page in your file.
  
` <!DOCTYPE html>  return
<html>  return
<head>  return
  <script src="https://rawgit.com/Pokemon9753/ymt/master/qrIcon4.js">  return
  </script>  return
	</head>  return
<body>  return
  <script>  return
    qrLink("left");  return
  </script>  return
</body>  return
</html>  return`
