# Qr Code
A widget that can be added to any website; creates a qr code that can be used to view the page in your file.
  ```html
<!DOCTYPE html>  
<html> 
<head> 
  <script src="https://rawgit.com/Pokemon9753/ymt/master/qrIcon4.js">
  </script> 
	</head> 
<body>
  <script> 
    qrLink("left"); 
  </script>
</body>
</html> 
```
